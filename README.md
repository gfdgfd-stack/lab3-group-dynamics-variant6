# Калькулятор дальности и максимальной высоты полета

## О проекте
Приложение предназначено для расчета параметров движения тела, брошенного под углом к горизонту без учета сопротивления воздуха. Пользователь вводит начальную скорость и угол броска, после чего программа вычисляет дальность полета и максимальную высоту.

Проект разрабатывается для повышения эффективности труда физиков, преподавателей и студентов. Приложение позволяет быстро получать результат без ручных вычислений и снижает вероятность ошибок при решении учебных и практических задач.

## Основные возможности
- ввод начальной скорости `v0`;
- ввод угла броска `α`;
- расчет дальности полета `L`;
- расчет максимальной высоты `H`;
- проверка корректности введенных данных;
- очистка формы для повторного расчета;
- краткая справка по формулам.

## Используемые формулы
При отсутствии сопротивления воздуха используются классические формулы:

- `L = (v0^2 * sin(2α)) / g`
- `H = (v0^2 * sin^2(α)) / (2g)`

где:
- `v0` — начальная скорость;
- `α` — угол броска к горизонту;
- `g = 9.81 м/с²`.
