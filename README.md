# Калькулятор дальности и максимальной высоты полета

## О проекте
Проект посвящен разработке приложения для расчета дальности полета и максимальной высоты тела, брошенного под углом к горизонту без учета сопротивления воздуха. Пользователь вводит начальную скорость и угол броска, после чего приложение автоматически выполняет расчет и выводит результат в удобной форме.

Программный продукт ориентирован на физиков, преподавателей и студентов. Он позволяет ускорить решение типовых задач по кинематике, уменьшить количество ручных вычислений и снизить вероятность ошибок при работе с формулами.

## Основные функции
- ввод начальной скорости;
- ввод угла броска;
- расчет дальности полета;
- расчет максимальной высоты;
- проверка корректности введенных данных;
- очистка формы;
- отображение краткой справки по формулам.

## Используемые формулы
- `L = (v0^2 * sin(2a)) / g`
- `H = (v0^2 * sin^2(a)) / (2g)`

где:
- `v0` - начальная скорость;
- `a` - угол броска;
- `g = 9.81 м/с^2`.
